# Church CRM

Church CRM is a Frappe/ERPNext custom app for nonprofit church management, inspired by CiviCRM.

## Features

- **Church Member Management** - Track members, households, relationships
- **Donation Tracking** - Tithes, offerings, building funds with ERPNext accounting integration
- **Membership Management** - Types, statuses, auto-renewal
- **Event Management** - Church events with registration and attendance
- **Pledge Management** - Pledge campaigns with payment schedules
- **Church Groups** - Ministries, committees, small groups, mailing lists
- **Reports** - Giving statements, donation summaries, membership reports

## Installation

```bash
bench get-app https://github.com/YOUR_USER/church_crm
bench --site your-site install-app church_crm
```
